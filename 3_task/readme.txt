Описание решения:

Сначала ищем все файлы с расширением .mp4 в текущем каталоге и его подкаталогах с помощью команды find.
Для каждого найденного видеофайла используем ffprobe для получения продолжительности в секундах.
С использованием awk складываем все продолжительности и выводим общую продолжительность в формате часы:минуты:секунды.
Повторяем поиск файлов .mp4 в каждой папке, вычисляем общую продолжительность видеофайлов в папке и выводим эту информацию.
Для проверки правильности подсчета длительности видеофайлов можно использовать команду ls для вывода списка видеофайлов в папке с их продолжительностью.
